# Linux Commands Fundamentals

A beginner-friendly 15-day roadmap for learning essential Linux terminal commands used in web development, servers, DevOps, cloud computing, and everyday system work.

## Who This Is For

This repository helps beginners learn Linux commands through short explanations, examples, exercises, and safe practice scripts.

## How to Learn

1. Open one day folder.
2. Read its `README.md`.
3. Type the examples in a Linux terminal.
4. Inspect `practice.sh` before running it.
5. Run it using `bash practice.sh`.
6. Complete the exercise without copying the answer.

## Safety

- Practice in a test directory, WSL, virtual machine, or container.
- Never run a command you do not understand.
- Be careful with `sudo`, `rm`, `chmod`, `chown`, disk tools, and service commands.
- `rm` normally does not use a recycle bin.
- Do not experiment as the `root` user.

## Repository Contents

Every day contains:

- `README.md` - notes, syntax, examples, and exercises
- `practice.sh` - a safe Bash practice script

The root `LINUX-COMMAND-MAP.md` provides an improved command map based on the original learning image.
