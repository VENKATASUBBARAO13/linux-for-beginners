# Linux and Terminal Introduction

## Goal

Understand Linux, terminals, shells, command syntax, and help.

## Commands and Concepts

| Command | Purpose | Example |
| --- | --- | --- |
| `echo` | Print text | `echo "Hello Linux"` |
| `whoami` | Show current username | `whoami` |
| `date` | Show date and time | `date` |
| `clear` | Clear the display | `clear` |
| `history` | Show command history | `history` |
| `man` | Read a command manual | `man ls` |
| `--help` | Show quick help | `ls --help` |

Command syntax usually follows `command [options] [arguments]`. Linux names are case-sensitive. Press `q` to leave most manual pages.
