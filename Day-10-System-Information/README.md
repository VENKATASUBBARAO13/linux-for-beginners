# System Information

## Goal

Identify the operating system, kernel, host, shell, and environment.

## Commands and Concepts

| Command | Purpose |
| --- | --- |
| `uname -a` | Show kernel/system details |
| `hostname` | Show computer name |
| `hostnamectl` | Show host and OS information |
| `uptime` | Show running time and load |
| `date` | Show or format date/time |
| `env` | List environment variables |
| `printenv HOME` | Print one variable |
| `which bash` | Show a command path |
| `type cd` | Explain command resolution |
| `cat /etc/os-release` | Show distribution details |
